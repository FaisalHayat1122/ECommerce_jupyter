"# Ecommerce-Store" 

🛒 Python E-Commerce Store

A simple console-based E-Commerce Store built using core Python concepts like:

✅ if, elif, else

✅ for loop

✅ while loop

✅ Functions

✅ Modules

✅ Lists

✅ User Input Handling

This project is perfect for beginners who want to understand how real-world shopping systems work using basic Python logic.

📌 Project Description

This E-Commerce Store allows users to:

Select product categories (Grocery, Electronics, Fruits & Vegetables)

View available products

Add products to cart

View cart items

Exit the store safely

The program runs in a loop until the user chooses to exit.

🧠 Concepts Used
🔹 Conditional Statements
if
elif
else

Used for:

Category selection

Product selection

Exit conditions

🔹 Loops

while loop

Used for:

This makes the code clean and professional.
